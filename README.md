# [Factory](https://factory-beige.vercel.app/)

Landing page for a factory.

## Features

- Carousel
- Slider
- Mobile layout

## Tech Stack

- Vite
- Sass
- Prettier
- Normalize.css
- Vercel
- Swiper
- Bootstrap

## How to use

1. Clone the repo

```bash
git clone https://github.com/SavitskayaKseniya22/factory.git
```

2. Change directory

```bash
cd factory
```

3. Install dependencies

```bash
npm install
```

4. Start the dev server

```bash
npm run dev
```
